session_snapshot.py

save_snapshot(headers, cookies, path, note)
— сохраняет текущие headers и cookies в JSON с мета-инфой (fetched_at, note).
— path по умолчанию "session_snapshot.json".

load_snapshot(path)
— загружает JSON snapshot, возвращает (headers, cookies, meta).
— meta содержит дату сохранения и заметку.

Использование:

save_snapshot(headers, cookies, note="initial fetch")
headers, cookies, meta = load_snapshot()


✅ Позволяет сохранять и восстанавливать сессию без повторного запроса, удобно для автоматизации.
